Cpp Project Manager

The Cpp Project Manager is a VSCode Extension for managing C++ projects.

Goals

  • Leverage CMake to easily link libraries with executables.
  • Provide templates for project files for faster development.
  • Add an `_internal` keyword for extended functionality.
  • Provide an easy interface to facilitate CTest test cases.
