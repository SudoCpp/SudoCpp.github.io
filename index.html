Cpp Project Manager

The Cpp Project Manger is a simple VSCode Extension to manage C++ projects by adding templates for classes, enums and tamplate classes. As well as, automatic linking of libraries and executables.

Simplex Toolkit

The Simplex Toolkit is a collection of classes to make C++ development easier (in a non-convensional means.) Typically C++ is not developed fully in an object oriented way, but rather a mix between different paradigms. This is uses a fully object oriented approach to make development faster and simplier by providing generic tooling.

Raspberry PI OS Dev
